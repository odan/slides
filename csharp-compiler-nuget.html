1. Was macht ein Compiler?

Der Compiler ist der Übersetzer zwischen dir und dem Computer.

  • ✓ Du schreibst lesbaren C#-Code
  • ✓ Computer versteht nur 0 und 1
  • ✓ Compiler übersetzt alles

2. Der C# Compiler: Roslyn

🔨

Roslyn

Der moderne C# Compiler von Microsoft. Übersetzt C# in Intermediate Language (IL).

⚡

4. Was sind NuGet Packages?

📦

Fertig entwickelte Bibliotheken von anderen Entwicklern

Beispiele: Newtonsoft.Json, Entity Framework, Serilog, AutoMapper...

Häufiges Problem

"The type or namespace name 'XXX' could not be found"

Der Compiler findet die NuGet-Bibliothek nicht.

Lösungen – Schritt für Schritt

  1. 1 Rechtsklick auf Projekt → Restore NuGet Packages
  2. 2 Clean Solution → Rebuild Solution
  3. 3 dotnet restore
  4. 4 Cache leeren: dotnet nuget locals all --clear

Zusammenfassung

→ Compiler (Roslyn) übersetzt deinen C#-Code in ausführbare Dateien

→ NuGet liefert fertige Bibliotheken

→ Bei Problemen: Restore + Clean + Rebuild
